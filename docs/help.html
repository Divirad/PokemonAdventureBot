Do you need help with this bot?

You started your adventure but you don't have any idea how to use this bot? Maybe this list of commands will help you! 

A non licensed Pokemonbot for Telegram. Made from real fans with the original Battle-Algorithms from the original games Red, Blue and Yellow.

/start
Starts the bot. If you aren't already registered you have to choose your starterpokemon. If you have not an account you will be asked if you want to delete your old account and create a new one.

/help
/menu
This command will print the menu

/pkmns
This command will print a list with your Pokemon in your team. You can change it with the computer. /com

/bag
will show you a list with your items.
